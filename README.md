# Python Flask App for Jenkins CI/CD

This is a simple Python Flask app that returns a message with **version** and **environment**.

## Environment Variables

- APP_VERSION: App version (default v1.0)
- ENVIRONMENT: Environment (default dev)

## Run Locally

```bash
pip install -r requirements.txt
python app.py
